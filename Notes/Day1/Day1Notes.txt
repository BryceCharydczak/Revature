CSS
Cascading style sheets

INLINE
    style attribute in element

INTERNAL
    written in head of html document in style tags

EXTERNAL
    link to a style sheet in the head of the document


Priority list of overriding:
1. inline
2. internal
3. external

higher ranking itmes in the priority list override the descending items.
